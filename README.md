# dou3d-ts

逗3D，基于 WebGL 的 3D 引擎，整体框架参考 AS3 时代的 Away3D 引擎，具体的实现参考了部分 H5 3D 引擎。

*本着以其看别人的引擎代码学习不如自己照着实现一个的想法，所以写了该引擎。*

* 该引擎主要是我自己为了了解3D底层运行原理而写，如果还有时间会扩充和完善示例。

[欢迎来我的博客，不定时发布 3D 学习心得。](https://www.cnblogs.com/hammerc/)
